# Release notes

## v1.3.0

Simplified for AI 1.3.0 expands the plugin from social and creative workflows
into broader Simplified operations while strengthening image-generation safety.

### Highlights

- Add `simplified-cli` guidance for authenticated API, media, social, comments,
  and project-management operations.
- Add `simplified-project-management` for boards, tasks, subtasks, dependencies,
  assignees, tags, custom fields, comments, and workflow reporting.
- Improve image-model selection with capability-first discovery, quality and cost
  tradeoffs, model-specific parameter handling, and baseline pricing guidance for
  GPT Image 2.
- Resolve brand assets by permanent asset ID and current file URL before passing
  them into model-specific image-generation fields.
- Refresh image-generation eval guidance and the hosted model snapshot notes.

### Compatibility

- The hosted Simplified MCP endpoint and OAuth setup are unchanged.
- Existing social, brand, project, creative, analytics, and review workflows remain
  available alongside the new skills.
